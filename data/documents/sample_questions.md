# Example questions:

## 1. In a Recurrent Neural Network (RNN), what is the primary role of the hidden state?
A. It serves as the network's memory by capturing information from previous time steps
B. It stores the final output of the entire sequence for downstream classification
C. It replaces the input at each time step with the predicted output from the previous step
D. It accumulates gradients during backpropagation without influencing forward computation
Answer: A


## 2. Which of the following is true?
A. BERT uses LSTMs
B. BERT is an autoregressive model
C. BERT is an encoder-based Transformer
D. BERT is a decoder-based Transformer
Answer: C


## 3. What is the main step when calculating the attention score in a seq2seq model?
A. Taking the dot product between the encoder and decoder hidden states
B. Averaging the encoder hidden states
C. Applying a softmax function to the decoder hidden states
D. Concatenating the encoder hidden states and passing them through a linear layer
Answer: A
